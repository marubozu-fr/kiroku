Settings → General · Chart Timeframes

A new Chart Timeframes card lets the user set two global defaults: the entry timeframe that pre-fills new trades, and the list of chart timeframes shown on every trade chart. The chip list is sorted by weight (D > W > H > M, then value descending) and shows a non-blocking orange warning above 8 timeframes. See README.md.
